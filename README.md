# kinklist-cn

## Web entry

The interactive app lives in [page/index.html](page/index.html). If you deploy the repository as a static site, opening the repo root will now redirect to that page through [index.html](index.html).

## Run locally

Serve the repository with any static file server, then open `page/index.html` or the repo root.

```bash
python3 -m http.server 8000
```

Then browse to:

- http://localhost:8000/
- http://localhost:8000/page/index.html
